# Job Board Application

A responsive Job Board web application built using React and Vite.  
Users can search jobs and filter them based on job type and location.

---

## Features

- Search jobs by:
  - Job Title
  - Company Name

- Filter jobs by Job Type:
  - Full Time
  - Contract
  - Internship

- Filter jobs by Location:
- Remote
- On-Site
- Hybrid
- WFO (Work From Office)

- Sort jobs alphabetically by title.
- Fast performance using Vite
- Clean and responsive UI

---

## Tech Stack

- React JS
- Vite
- JavaScript (ES6)
- CSS
